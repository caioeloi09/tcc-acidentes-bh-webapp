# Backend - REST API

## Technologies

- Kotlin 1.9+
- Spring Boot 3.x
- Spring Data JPA
- SQLite
- Maven

## How to Run

```bash
mvn spring-boot:run
```

API available at: `http://localhost:8080`. Hibernate does not own the schema
(`ddl-auto: none`); it reads the SQLite file produced by the migration script.

## Testing

```bash
mvn test
```

Covers `AccidentService` (aggregations) and `AccidentController` (endpoints).

## Main Endpoints

- `GET /api/accidents` вЂ” list accidents with optional filters
- `GET /api/accidents/{id}` вЂ” details of a specific accident
- `GET /api/statistics` вЂ” aggregated statistics (filters: `year`, `district`, `type`)
- `GET /api/map` вЂ” accidents with coordinates for the map
- `GET /api/ml/forecast` В· `GET /api/ml/severity` В· `GET /api/ml/heatmap` вЂ”
  proxied to the [ML service](../ml-service/) (`ml.service.url`, default
  `http://localhost:8000`); returns `503` if it is offline

See the full [REST API reference](../docs/api/README.md).
